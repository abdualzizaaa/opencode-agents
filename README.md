## 📖 About the Project | عن المشروع

<div dir="rtl">

**OpenCode Agents** هي مجموعة شاملة تحتوي على أكثر من **200 عامل ذكي** مصممين للاستخدام مع أنظمة البرمجة والتسويق والإدارة وتطوير المنتجات. كل عامل مخصص لمهام محددة ويعمل كمساعد ذكي متخصص في مجاله.

</div>

<div dir="ltr">

**OpenCode Agents** is a comprehensive collection of **200+ AI agent definitions** designed for use with coding, marketing, project management, and product development systems. Each agent is specialized for specific tasks and works as an intelligent assistant in its domain.

</div>

## 🔧 Agent Format | صيغة العامل

<div dir="rtl">

كل ملف عامل يتبع الصيغة التالية:

```markdown
# اسم العامل

## الدور والهوية
- **الاسم:** اسم العامل
- **الدور:** وصف الدور
- **الخبرات:** مجالات الخبرة

## المهمات الرئيسية
- وصف المهمات التي ينفذها العامل

## طريقة العمل
- كيفية تنفيذ المهام

## الأدوات المطلوبة
- الأدوات والتقنيات المستخدمة
```

</div>

<div dir="ltr">

Each agent file follows this format:

```markdown
# Agent Name

## Role & Identity
- **Name:** Agent name
- **Role:** Role description
- **Expertise:** Areas of expertise

## Key Tasks
- Description of tasks the agent performs

## Workflow
- How tasks are executed

## Required Tools
- Tools and technologies used
```

</div>
